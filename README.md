## messenger

![Messenger](https://i.ibb.co/dgQSTcQ/messenger-gallery.png)

Приложение для обмена сообщениями. Написано в рамках курса Яндекс.Практикум Мидл фронтенд разработчик.

#### Доступные роуты:

- / - all users
- /404 - all user
- /500 - all user
- /signin - guest user
- /signup - guest user
- /messenger - auth user
- /messenger/:id - auth user
- /profile - auth user
